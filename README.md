## How to Contribute

Please add your own resources on video game preservation to the list above. Order does not dictate preference. Appropriate additions may include:

- metadata schemas
- preservation plans
- case studies

For help contributing to this repository or other issues, please [submit a New Issue](https://github.com/amiaopensource/analog-inspection/issues/new) with your question and we'll get back to you.
